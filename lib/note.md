# Photo Calendar App (사진 달력)

---

## 프로젝트 목표
- 날짜별로 사진 저장
- 달력에서 썸네일 표시
- 날짜 클릭 시 상세 사진 보기

---

## 기능 구조
- 사진 불러오기 -> 앨범설정창 (제목, 날짜변경, 메모)
- 앨범 내에 사진 추가

---

## 아이디어
- 클라우드 백업 (Firebase?)
- 필터 기능 추가
- 썸네일 앨범 내에서 선택한 사진으로
- 텍스트 띄우기 / 위버스 연동 캡쳐
